使用 hexo 搭建博客

Posted on 2015-06-07 | Edited on 2018-05-04 |

简单地记录一下本站建设过程

#配置 hexo

  1. 安装 node.js
  2. 使用淘宝镜像加速

  3. 安装 hexo

#配置 git

  1. 配置 ssh 公钥
  2. 在 github 或 gitcafe 建立 pages

#建立博客

  1. 初始化

  2. 配置 _config.yml

  3. 设置主题，我用的是 NexT
  4. 设置多说
  5. 生成静态文件 hexo g
  6. 本地服务器调试 hexo s
  7. 发布 hexo d

#写博客

  1. 新建文章 hexo n "文章标题"
  2. 使用 markdown 编辑器写作
  3. 生成发布 hexo d -g

#个人域名

  1. 购买域名，namecheap 或 godaddy
  2. 设置 github 的 CNAME，或 gitcafe 的 pages
  3. dnspod 绑定

#可选优化

  1. 多说速度不稳定，使用七牛云存储做镜像来提速
  2. 使用 dnspod 分流国内外流量
  3. 优化字体加载
